## General Style Guidelines

For all images, use these modifiers:
- "minimalist design"
- "Apple aesthetic" or "Apple style"
- "clean"
- "professional"
- "modern"
- "elegant"
- "high quality"
- "white background" (unless specified otherwise)

Avoid:
- Cluttered designs
- Too many colors
- Complex backgrounds
- Cartoonish styles
- Overly technical diagrams

---

## Image Sources (Free Alternatives)

If not using AI generation, consider:
1. **Unsplash** - Search "security", "lock", "authentication"
2. **Pexels** - Free stock photos
3. **Flaticon** - Free icons (with attribution)
4. **The Noun Project** - Icon library
5. **Freepik** - Free vectors and graphics

---

## Recommended Image Sizes

- Cover: 1200x1600px (or higher)
- Page images: 800x600px minimum
- Icons: 200x200px minimum
- Product shots: 1000x1000px

All images should be high resolution (300 DPI) for print quality.
